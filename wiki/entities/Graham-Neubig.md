# Graham Neubig

> 卡内基梅隆大学（CMU）语言技术研究所教授，多语言 NLP 评测领域的重要贡献者，PromptBench 和 SWE-bench 等评测工具的参与开发者。

## 基本信息

- **所属机构**：卡内基梅隆大学（Carnegie Mellon University）语言技术研究所（LTI）
- **职位**：副教授
- **研究方向**：自然语言处理、多语言 NLP、低资源语言、代码生成、NLP 评测
- **背景**：曾在奈良先端科学技术大学院大学（NAIST）和日本 RIKEN 从事研究

## 评测领域主要贡献

**多语言 NLP 评测**：Graham Neubig 在多语言和低资源语言 NLP 评测上贡献突出，其团队开发和参与了多个多语言评测数据集，推动了评测资源从英语单语向多语言覆盖的扩展。

**SWE-bench（2023）**：Neubig 团队参与开发的 SWE-bench 是评测 LLM 解决真实 GitHub Issues（软件工程问题）能力的重要基准。SWE-bench 包含 2,294 个来自真实开源项目的问题，要求模型生成能通过测试用例的代码补丁，成为评测 LLM 实际软件工程能力的行业标准。

**PromptBench**：参与开发的 PromptBench 评测框架，测试 LLM 对于对抗性提示和提示扰动的鲁棒性，推动了"提示鲁棒性"作为 LLM 评测维度的研究。

**CodeAct / 代码生成评测**：在 LLM 代码生成和 Agent 行动能力评测方面有系统性研究，包括使用代码作为 Agent 行动空间的评测框架。

**低资源语言评测数据集**：Neubig 团队构建了多个低资源语言的 NLP 评测数据集，填补了现有评测体系在非主流语言上的空白。

## 代表性工作

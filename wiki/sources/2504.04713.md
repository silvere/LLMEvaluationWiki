# Sequential-NIAH（2504.04713）

> 2025-04 发表的 [[NIAH]] 升级 benchmark：把单针召回扩展到「按顺序提取多个针」，覆盖 8K-128K 上下文长度。提供 14K 训练样本（基于合成 + GPT-4 生成），是当前 long-context LLM「sequential reasoning over long context」评测的代表工作之一。

## 设计

- **任务**：在 haystack 中按指定顺序提取多个 needle
- **上下文长度**：8K / 16K / 32K / 64K / 128K
- **训练集**：14,000 样本，可用于 SFT
- **指标**：sequential recall accuracy + order correctness

## 与 NIAH 升级关系

- [[NIAH]] v1：单针、随机深度
- Multi-Needle NIAH：多针、同时召回
- **Sequential-NIAH**：多针 + 顺序约束（更接近真实下游任务）
- [[RULER]]：13 个长上下文子任务，含 sequential variants

## 影响

- 揭示即使是顶级 long-context LLM 在 sequential needle 上仍显著退化
- 是 2025 长上下文模型 ablation 的常见对照
- 推动 long-context training data 设计向「sequential reasoning」倾斜

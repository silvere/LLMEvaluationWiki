# Tianxiang Sun（孙天祥）

> Analemma 创始人 CEO + 助理教授；复旦大学 2024 CS PhD（导师邱锡鹏 + 黄萱菁），中国 LLM 后训练 / RL 方向的青年研究者，参与复旦 MOSS 等开源模型。

## 基本信息

- **所属机构**：Analemma（创始 CEO）· 高校（assistant professor）·复旦大学 校友（NLP Lab）
- **职位**：Founder & CEO, Analemma · Assistant Professor
- **学历背景**：复旦大学 PhD（CS，2024，advisor 邱锡鹏 [Xipeng Qiu] + 黄萱菁 [Xuanjing Huang]）
- **链接**：[主页](https://txsun1997.github.io/)

## 评测领域主要贡献

**MOSS 系列开源模型**：作为复旦 NLP Lab 重要成员，参与 MOSS（中国早期开源大模型代表之一）项目，包括其训练与多任务评测体系（[[C-Eval]] / [[CMMLU]] / [[AGIEval]] 等中文评测的早期实测）。

**Post-Training & RL 评测**：研究兴趣聚焦 RL 后训练对预训练大模型在各场景的能力提升与评测；这条线对 [[process-reward-model]] / [[reward-modeling]] / [[Reward Hacking 评测]] 等议题在中文圈的研究有方法学贡献。

**Prompt Tuning / Parameter-Efficient Tuning 评测**：博士期间多篇 prompt tuning / PEFT 评测论文，影响中文 LLM 适配评测的实践协议。

## 代表性工作

- MOSS 系列模型（复旦）
- Post-training RL 评测系列
- Prompt tuning / PEFT 系列

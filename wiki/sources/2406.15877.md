# BigCodeBench: Benchmarking Code Generation with Diverse Function Calls and Complex Instructions

## 论文概述

BigCodeBench 是专注于评测 LLM 代码生成能力（特别是函数调用和复杂指令执行）的综合性代码评测基准。该基准弥补了 HumanEval 等早期代码基准在任务复杂度和库调用多样性上的不足。

## 数据集设计

**任务规模**：共包含1,140道编程任务，覆盖多个主流 Python 库（如 pandas、numpy、requests、sklearn 等139个库）。

**任务类型**：
- **BigCodeBench-Complete**：给定函数签名和文档字符串，补全函数实现
- **BigCodeBench-Instruct**：仅给定自然语言指令，要求从头实现功能

**复杂度**：每道题平均需要调用5个以上不同的库函数，相比 HumanEval（单一函数实现）难度显著提升。

**测试评估**：使用单元测试（Unit Tests）进行功能正确性验证，采用 pass@k 指标。

## 主要发现

- GPT-4 在 BigCodeBench 上的 pass@1 约为61%，揭示即使最强模型在复杂多库调用任务上仍有显著局限
- Instruct 格式比 Complete 格式更具挑战性（约低8-10个百分点）
- 开源代码模型（如 DeepSeek-Coder、CodeLlama）在该基准上与商业模型差距较小

## 学术影响

BigCodeBench 推动了代码生成评测向"真实编程任务复杂度"对齐，成为继 HumanEval 和 MBPP 之后代码 LLM 评测的重要基准之一。

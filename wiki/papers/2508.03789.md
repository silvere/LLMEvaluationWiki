# HPSv3: Towards Wide-Spectrum Human Preference Score

> [[HPSv2]] 续作（CUHK MMLab + MizzenAI，ICCV 2025）。**HPDv3 数据集 1.08M 文图对 + 1.17M 标注对比**，VLM-based preference model + uncertainty-aware ranking loss + Chain-of-Human-Preference 迭代精炼。

## 论文核心贡献

1. **HPDv3 数据集**：1.08M 文图对 + 1.17M pairwise 比较（v2 是 433K 对比）
2. **VLM-based preference model**：相比 v2 的 CLIP-based 更强语义理解
3. **Uncertainty-aware ranking loss**：对标注分歧大的对降权
4. **Chain-of-Human-Preference (CoHP)**：迭代式 prompt 改写 → 生成 → 评分回路，作为图像质量改进信号

## 演进路径

- v1 (2023)：基础 CLIP-based preference
- **v2 (2023, arxiv:2306.09341)**：HPDv2 433K 对比，业界主流
- **v3 (2025, 本文)**：1.17M 对比 + VLM-based + CoHP

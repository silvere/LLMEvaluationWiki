# SWE-Lancer: Can Frontier LLMs Earn $1 Million from Real-World Freelance Software Engineering?

> OpenAI 2025-02 推出的**真实自由职业软件工程任务评测**。1400+ 个 Upwork 任务，**真实 payout 总额 $1M**，覆盖 $50 bug fix 到 $32K 功能实现。Claude-3.5-Sonnet 仅赚到 $400K，证明 frontier 模型在真实软工经济价值上仍有显著差距。

## 论文核心贡献

1. **真实经济价值评测**：1400+ Upwork 任务 + 真实 $1M payout
2. **两类任务**：
   - Independent engineering（$50-$32K bug + feature）
   - Managerial（model 在工程方案间选择）
3. **三重验证 end-to-end 测试**：每个任务 3 位资深工程师验证 test case
4. **Claude-3.5-Sonnet 仅 $400K / $1M**：frontier 模型经济竞争力可量化
5. **开源 Docker + SWE-Lancer Diamond 评测 split**

## 评测圈意义

- 是 [[SWE-bench-Verified]] / [[TheAgentCompany]] 之后**面向经济价值**的下一代软工 agent 评测
- 「能赚多少钱」是比「pass 多少 test」更接近真实部署的度量
- Diamond split 适合每月轮换 SOTA 跟踪

## 限制 / pitfall

- 任务来自 Upwork 历史 → **数据污染风险**（公开仓库 + 时间敏感）
- 仅前端 / web 任务，少后端 / 系统工程
- Managerial 任务依赖原始招聘 manager 的决策，存在主观性

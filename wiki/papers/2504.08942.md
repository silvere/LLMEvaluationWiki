# AgentRewardBench: Evaluating Automatic Evaluations of Web Agent Trajectories

> ServiceNow Research + McGill + Mila 2025-04 推出的 **web agent 轨迹自动评测的元评测**。**1302 轨迹横跨多个 benchmark + LLM**，证明「没有单一 LLM 在所有 benchmark 上 dominate」+ 「传统 rule-based eval 会低估 web agent 成功率」。

## 论文核心贡献

1. **首个 web agent 轨迹评测的 meta-benchmark**：评估「评估器」自己
2. **1302 轨迹数据集**：跨 [[BrowserGym]] / [[WebArena]] / [[VisualWebArena]] 等
3. **揭示 rule-based eval 低估问题**：自动规则评分系统性 underreport
4. **跨 LLM judge 多样性**：没有「全能 judge」，需多模型聚合
5. **为「LLM-as-judge for agents」铺路**：[[dimension-K-judge]] 的 agent 子档

## 评测圈意义

- 是 [[JudgeBench]] / [[RewardBench]] 之外**面向 agent 子赛道**的 meta-eval
- 解决「agent benchmark 评分本身可信吗」这一上层问题
- 与 [[OSWorld-Human]] 都是 2025 年「评测评估器」浪潮的代表

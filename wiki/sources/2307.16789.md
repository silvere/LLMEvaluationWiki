# ToolLLM: Facilitating Large Language Models to Master 16000+ Real-world APIs

## 论文概述

ToolLLM 是清华大学等机构提出的工具使用能力训练与评测框架，发表于2023年7月。该工作系统解决了如何让 LLM 学习使用大规模真实世界 API 的问题，并构建了相应的评测基准 ToolBench。

## 数据集与评测基准

**ToolBench**：基于 RapidAPI 平台收集了超过16,000个真实 API，涵盖49个类别。通过 ChatGPT 生成工具使用场景，构建了包含多步 API 调用的指令-解决方案对数据集。

**评测任务类型**：
- 单工具单步任务（I1）
- 单工具多步任务（I2）
- 跨工具多步任务（I3）

**评测指标**：
- 通过率（Pass Rate）：基于 ChatGPT 裁判评估解决方案是否完成了用户指令
- 优于 ChatGPT 率（Win Rate）：与 ChatGPT baseline 的逐对比较

## 核心方法

**DFSDT（深度优先搜索决策树）**：提出基于决策树搜索的 API 调用规划方法，相比线性规划更好地处理多步工具使用中的回溯和错误恢复场景。

**ToolLLaMA**：基于 LLaMA 微调的工具使用模型，在 ToolBench 上达到接近 ChatGPT 的工具使用能力。

## 学术影响

ToolLLM 推动了 LLM 工具使用能力评测的系统化，其大规模真实 API 评测范式被后续工具代理研究广泛采纳。该工作也引发了关于如何定义和测量"工具使用能力"这一 LLM 核心能力维度的深入讨论。

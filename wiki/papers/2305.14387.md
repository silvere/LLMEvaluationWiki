# AlpacaFarm: A Simulation Framework for Methods that Learn from Human Feedback

## 论文概述

AlpacaFarm 是斯坦福大学提出的 RLHF（来自人类反馈的强化学习）研究仿真框架，发表于2023年5月。该工作的核心贡献是提供了一套低成本、可复现的 RLHF 研究实验环境，解决了 RLHF 研究依赖大规模真实人工标注的成本瓶颈问题。

## 主要方法

**LLM 仿真人工反馈**：AlpacaFarm 使用 GPT-3.5/GPT-4 等强语言模型模拟人类评估者，对指令跟随任务的模型输出进行偏好比较。研究表明，该 LLM 仿真偏好与真实人工偏好具有较高一致性，可作为 RLHF 训练的替代信号。

**完整 RLHF 管线实现**：框架实现了从偏好数据采集、奖励模型训练到 PPO 强化学习微调的完整流程，并在单一框架内对比了 SFT（监督微调）、RLHF（PPO）和 RLAIF（AI 反馈）等方法的效果差异。

**评测方法论**：提出了基于 Pairwise Comparison（逐对比较）的自动化评测流程，用 GPT-4 作为裁判模型评估不同方法训练的模型在开放指令任务上的相对质量。

## 主要发现

- LLM 仿真偏好在方法排名上与真实人工偏好高度一致（相关系数 > 0.90）
- RLHF（PPO）在指令跟随能力上显著优于单纯的 SFT
- 奖励模型质量是 RLHF 效果的关键瓶颈

## 学术影响

AlpacaFarm 推动了 RLHF 研究的民主化，其 LLM-as-Judge 评测方法论被广泛采纳，也引发了关于 AI 评测中偏好对齐可靠性的深入讨论。

# Peter Clark

Peter Clark 是 Allen Institute for AI（AI2）的高级研究员和项目负责人，长期专注于常识推理评测和知识驱动的 NLP 研究。他是 ARC（AI2 Reasoning Challenge）基准的主要作者之一，对 LLM 科学推理能力评测有奠基性贡献。

## 核心研究领域

**常识推理与知识问答**：Clark 的研究核心是设计能真正测试机器推理能力的评测任务，而非仅测试模式匹配能力。他主张评测应要求模型展示对世界知识的实际理解和推理应用，而非通过统计记忆作答。

**ARC 基准（AI2 Reasoning Challenge）**：Clark 是 ARC 数据集的主要创建者之一。ARC 包含来自美国小学至高中科学考试的问题，分为"简单集"（Easy Set）和"挑战集"（Challenge Set）两部分，挑战集专门筛选了当时最先进的 NLP 系统无法正确回答的题目。ARC Challenge Set 成为衡量 LLM 常识科学推理能力的重要基准，被 MMLU、Open LLM Leaderboard 等主流评测体系采纳。

## 其他评测贡献

Clark 还参与了多个常识推理相关基准的构建工作：

- **QASC**（Question Answering via Sentence Composition）：评测需要多跳推理的科学问答
- **Winogrande**：大规模共指消解和常识推理数据集（联合贡献）
- **ProofWriter**：评测 LLM 进行多步逻辑推理和证明生成的能力

## 知识与推理研究观点

Clark 是 AI 领域"知识导向"评测观的代表人物，强调语言理解评测不应局限于语言形式层面，而必须检验模型对概念知识的实际掌握程度。他的研究对推动 LLM 评测从"语言流畅性"向"认知能力深度"转型具有重要意义。

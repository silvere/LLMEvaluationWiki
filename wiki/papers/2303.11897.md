# TIFA: Accurate and Interpretable Text-to-Image Faithfulness Evaluation with Question Answering

> UW + AI2 2023-03 经典论文（ICCV 2023）。**首个用 VQA 评测 T2I faithfulness 的自动方法**：LM 生成 (问题, 答案) 对 → VQA 模型在生成图上回答 → 准确率 = faithfulness。无需参考图像，可解释。

## 论文核心贡献

1. **VQA-based faithfulness**：用 LM 把 prompt 拆解成可验证的 QA pair
2. **TIFA v1.0 benchmark**：4000 文本 prompt × 25000 questions × 12 类别
3. **揭示 T2I 模型在 counting / spatial relation / multi-object 上的薄弱**
4. **可解释**：能指出「哪一个 QA 失败 → 哪个细节没画对」

## 影响

- 是 [[VQAScore]] / [[GenAI-Bench]] / [[GenEval]] / [[Soft-TIFA]]（即 [[GenEval 2]] 的 judge）的奠基论文
- 业界 T2I 评测从「整图 CLIP score」转向「原子级 QA」即源自此文

# Ion Stoica

> UC Berkeley EECS 教授；Sky Computing Lab 主任。Apache Spark / Ray / Databricks / Anyscale 等核心 ML 基础设施联合创始人。Chatbot Arena 所在的 LMSYS 研究组的学术 sponsor。

## 基本信息

- **所属机构**：UC Berkeley EECS（2001 起，从 CMU 转入）·Sky Computing Lab Director ·Databricks Co-founder ·Anyscale Co-founder
- **职位**：Professor of EECS, UC Berkeley
- **学历背景**：CMU PhD（CS）
- **链接**：[Berkeley 主页](https://people.eecs.berkeley.edu/~istoica/) · [Wikipedia](https://en.wikipedia.org/wiki/Ion_Stoica) · [Google Scholar](https://scholar.google.com/citations?user=vN-is70AAAAJ)

## 评测领域主要贡献

**LMSYS / Chatbot Arena 学术 sponsor**：作为 Berkeley Sky Computing Lab 主任，孵化了 Lianmin Zheng、Wei-Lin Chiang、Hao Zhang 等核心团队启动 [[LMSYS-Org]] 的 Chatbot Arena 项目。其学术支持与基础设施（vLLM 等）是 Arena 评测可扩展性的关键背景。

**vLLM 服务系统**：Sky Lab 孵化的 vLLM 已成为 LLM serving 的事实标准，间接定义了「fast pass@1 evaluation」「multi-batch eval」「serving-aware benchmark」的工程实践。

**AI 基础设施奠基**：作为 Apache Spark / Ray / Databricks / Anyscale 的联合创始人，长期推动 ML 系统 + 数据基础设施。这些工具是当前 [[HELM]] / [[OpenCompass]] / [[LM-Evaluation-Harness]] 等评测平台的底层支撑。

**Sky Computing 评测愿景**：提出 Sky Computing 概念（多云、跨 region 统一 ML 计算），为未来「跨 provider 模型评测一致性」议题提供概念框架。

# Fudan NLP Lab（复旦大学自然语言处理实验室）

> 复旦大学 NLP 研究 group，由 [[Xipeng-Qiu|邱锡鹏]] 教授带领；MOSS 开源大模型主力发布团队。与 Shanghai Innovation Institution（SII）合作组建 OpenMOSS Team，是中国最早一批做开源对话大模型的学术力量。

## 基本信息

- **所属机构**：Fudan University 计算机科学学院
- **关键 PI**：[[Xipeng-Qiu]]（邱锡鹏）、Xuanjing Huang（黄萱菁）等
- **核心项目**：MOSS 系列开源大模型；OpenMOSS Team（与 SII / MOSI Intelligence 合作）
- **链接**：[OpenMOSS GitHub](https://github.com/OpenMOSS/MOSS) · [HuggingFace](https://huggingface.co/fnlp)

## 在 LLM 评测圈的贡献

**MOSS 系列开源**：2023-02 发布首版 MOSS（16B 对话模型），是中国早期开源大模型代表，其在 [[C-Eval]] / [[CMMLU]] / [[AGIEval]] 等中文评测的实测结果是国内 LLM 评测体系建立的早期 baseline。

**评测人才培养**：培养出 [[Tianxiang-Sun|孙天祥]] 等专注 RL post-training 评测的学者，扩散到工业界 LLM 评测一线。

**Tool-Augmented LLM 评测早期实践**：MOSS 早期就支持工具调用，影响后续 tool-use evaluation 在中文场景的落地实践。

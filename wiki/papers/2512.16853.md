# GenEval 2: Addressing Benchmark Drift in Text-to-Image Evaluation

> Meta FAIR + UW + AI2 2025-12 推出的 [[GenEval]] 续作。**揭示 GenEval v1 已与人类判断 drift 17.7 pt（绝对误差）**，提出 **Soft-TIFA** 原子级 VQA judge 替代，覆盖更细粒度的对象 / 属性 / 关系 / 计数。

## 论文核心贡献

1. **暴露 benchmark drift 现象**：v1 上 90+ 分模型，在人类标注上仅 72%
2. **Soft-TIFA judge**：把 [[TIFA]] 的视觉原子判断软化为概率分布，比 [[VQAScore]] 更抗 drift
3. **更高 compositionality**：物体 × 属性 × 关系 × 计数的组合空间显著扩大
4. **预期更长寿命**：原子化判断与人类感知耦合更稳，难以被「学到目标 metric」的模型绕过

## 限制 / pitfall

- 仍依赖底层 VLM 的视觉理解能力（VLM 弱则 judge 弱）
- Soft-TIFA 的 atomic-level 拆解对长 prompt 可能不完整
- 与 [[GenEval]] v1 分数不可直接比较

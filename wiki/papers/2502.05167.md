# NoLiMa: Long-Context Evaluation Beyond Literal Matching

> Adobe Research + LMU Munich 2025-02 推出的**长上下文语义匹配评测**（ICML 2025）。突破 [[NIAH]] 经典「字面命中」限制——**强制 needle 与 question 最小字面重叠**，逼模型做语义检索。13 个 128K+ 模型测试，**11 个在 32K 上跌破 50% 基线**，GPT-4o 从 99.3% 掉到 69.7%。

## 论文核心贡献

1. **超越字面匹配的 needle**：question 与 needle 词汇几乎不重叠
2. **13 模型 × 128K+ 实测**：覆盖主流长上下文模型
3. **揭示 attention 在长 context 下的语义检索短板**：传统 NIAH 高估了模型能力
4. **GPT-4o 性能从 99.3% 掉到 69.7%**：是 [[RULER]] / [[HELMET]] 之外的强补充

## 评测圈意义

- 是 [[NIAH]] 「字面命中」时代的终结者
- 与 [[RULER]] / [[HELMET]] / [[InfiniteBench]] 互补：NoLiMa 强调语义，RULER 强调多任务
- 「真长上下文 = 语义检索 + 多跳推理」成为业界共识

## 限制 / pitfall

- needle 设计仍人工，规模化有局限
- 32K 以上的「语义难度」尚无统一度量
- 与下游应用（如 RAG）的对应关系仍需研究

# Tsinghua NLP Lab（清华大学自然语言处理实验室 / THUNLP）

> 清华大学 NLP / 信息检索研究 group，由刘知远（Zhiyuan Liu）、孙茂松（Maosong Sun）等老师带领。OpenBMB 开源生态发起方之一，培养出 [[Yujia-Qin]] 等中国 tool-use / agent 评测主力研究者。

## 基本信息

- **所属机构**：清华大学 计算机科学与技术系
- **关键 PI**：[[Maosong-Sun|孙茂松]]、刘知远（Zhiyuan Liu）、Le-Yang Hou 等
- **关联开源**：OpenBMB（BMTrain / BMInf / BMCook / [[ToolBench]] / MiniCPM 等）
- **链接**：[OpenBMB GitHub](https://github.com/OpenBMB)

## 在 LLM 评测圈的贡献

**ToolLLM / ToolBench**：[[Yujia-Qin]] 等的 [[ToolBench]] 项目是 tool-use evaluation 的奠基工作之一，把工具调用评测推到 16K+ 真实 API 规模。后续 StableToolBench 等延续此范式。

**MiniCPM 小模型评测**：OpenBMB 推出的 MiniCPM 系列在 small-model 评测维度推动「小模型也能在 [[MMLU]] / [[BBH]] 等 benchmark 上达到与大模型可比性能」的方法学讨论。

**预训练优化工具链评测**：BMTrain 等开源工具影响 LLM 训练 / 评测的工程化实践。

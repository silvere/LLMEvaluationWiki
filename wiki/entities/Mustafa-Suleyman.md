# Mustafa Suleyman

> Microsoft AI CEO（2024-03 起）；DeepMind 联合创始人（2010，与 Demis Hassabis、Shane Legg）；Inflection AI 联合创始人（Pi 个人化 AI）。2026-03 重组后聚焦 Microsoft AI 的 frontier 模型与「Human Superintelligence (HSI)」战略。

## 基本信息

- **所属机构**：Microsoft AI（CEO，2024-03 起）·前 Inflection AI 联合创始人 ·前 DeepMind / Google
- **职位**：CEO, Microsoft AI
- **学历背景**：Oxford 辍学（哲学 / 神学 / 政治）
- **链接**：[Wikipedia](https://en.wikipedia.org/wiki/Mustafa_Suleyman) · [Microsoft 公告](https://blogs.microsoft.com/blog/2024/03/19/mustafa-suleyman-deepmind-and-inflection-co-founder-joins-microsoft-to-lead-copilot/)

## 评测领域主要贡献

**DeepMind 早期评测方向**：2010 与 Demis Hassabis、Shane Legg 共同创立 DeepMind，作为 Chief Product Officer 把 DQN / AlphaGo / AlphaFold 等 system 推向工业级评测。DeepMind 在 Go / 蛋白质折叠 / 游戏环境评测上的方法学影响后续 LLM 评测对 "non-text capability" 的关注（[[CASP]] / Go evaluation 类）。

**Inflection AI / Pi 个性化评测**：2022-2024 在 Inflection 推动「personality-aware evaluation」——把对话评测从「正确性」扩展到「情感对齐、个性一致性」。Pi 对人格化 LLM 评测路径有奠基意义。

**Microsoft AI Copilot 评测体系（2024+）**：作为 CEO 主导 Copilot 系列评测体系——把企业级 productivity 评测（[[MMLU]] / [[HumanEval]] / 客户使用真实 task）整合到产品决策。

**Human Superintelligence (HSI) 战略**：2026-03 重组后提出 HSI 框架——评测目标从「benchmark 通过」转向「is the AI working for humanity」的对齐性评测——这是 [[capability-elicitation]] / [[red-teaming]] 等评测在工业 narrative 层的重要演化。

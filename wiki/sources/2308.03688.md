# AgentBench: Evaluating LLMs as Agents

## 论文概述

AgentBench 是清华大学和俄亥俄州立大学等机构联合提出的综合性 LLM 代理评测基准，发表于2023年8月。该基准旨在系统评测大型语言模型作为自主代理（Autonomous Agent）完成现实世界任务的能力，填补了传统 NLP 基准与实际代理应用场景之间的评测空白。

## 评测设计

AgentBench 涵盖8个不同的代理任务环境：

1. **操作系统（OS）**：在 Linux 终端中执行命令行任务
2. **数据库（DB）**：数据库查询和操作
3. **知识图谱（KG）**：在知识图谱上进行推理查询
4. **数字卡牌游戏（DCG）**：策略性卡牌游戏决策
5. **横向思维谜题（LTP）**：需要多轮提问和推理的谜题
6. **家庭场景（ALFWorld）**：基于文本的家庭环境任务执行
7. **网络浏览（WebShop）**：电商网站上的商品搜索和购买
8. **Mind2Web**：网页操作任务

## 主要发现

- GPT-4 在所有任务中表现最佳，但与人类水平仍有显著差距
- 开源模型与商业模型之间存在明显能力鸿沟，特别是在需要多步规划的任务上
- LLM 代理在操作系统和代码执行类任务上表现相对较好，在需要复杂世界知识的任务上表现较弱
- 模型的指令跟随能力直接影响代理任务成功率

## 学术影响

AgentBench 系统化了 LLM 代理能力评测的方法论，推动了代理基准设计的标准化。其多环境、多任务的评测框架被后续代理评测研究广泛参考。

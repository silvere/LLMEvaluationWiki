# Andrej Karpathy

> 前特斯拉 AI 总监、前 OpenAI 研究员，LLM 教育与科普贡献者，以清晰解释 LLM 工作原理著称，llm.c 和 nanoGPT 的作者。

## 基本信息

- **所属机构**：独立研究者（前 OpenAI、前 Tesla）
- **研究方向**：深度学习、计算机视觉、大语言模型、AI 教育
- **学历背景**：斯坦福大学博士（李飞飞指导）

## 评测领域主要贡献

**LLM 教育与理解**：Andrej Karpathy 的主要贡献是将 LLM 的工作原理以可理解的方式向公众和研究者解释。其 YouTube 系列视频（"Neural Networks: Zero to Hero"）和相关推文，帮助大量研究者建立了对 LLM 能力与局限的直觉理解，间接影响了评测议题的讨论方向。

**nanoGPT**：开源的最简 GPT 实现（约 300 行 Python），使研究者可以在最小规模上复现 GPT 类模型的训练和评测过程，成为理解和评测 LLM 基础行为的教学工具。

**llm.c**：用纯 C/CUDA 实现的 GPT-2 训练代码，展示了 LLM 实现的极简可能性，为评测 LLM 训练效率和系统性能提供了参照基准。

**"Software 2.0"概念**：提出了"神经网络作为软件"的概念框架，影响了 AI 系统评测从传统软件测试视角向统计性能评测视角的转换讨论。

**特斯拉 Autopilot 评测**：在特斯拉任职期间，主导了基于摄像头的自动驾驶视觉系统的评测框架，其大规模真实世界评测方法对 AI 系统评测实践有借鉴意义。

**"LLM OS"概念**：提出 LLM 作为操作系统内核的类比框架，为 AI Agent 评测的维度划分提供了概念工具。

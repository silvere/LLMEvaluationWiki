# Meta Llama 3 技术报告

## 报告概述

本报告是 Meta AI 于2024年4月发布的 Llama 3 系列模型技术报告，详细介绍了 Llama 3（8B 和 70B 参数版本）的训练方法、评测结果和技术细节。Llama 3 是目前最广泛使用的开源大型语言模型系列之一。

## 模型技术特点

**训练数据**：Llama 3 使用超过15万亿 tokens 的训练数据，相比 Llama 2 规模大幅提升，涵盖更多代码、数学和多语言内容。词表规模从32,000扩展到128,256，改善了非英语语言的处理效率。

**架构改进**：采用 Grouped Query Attention（GQA）提升推理效率，支持更长的上下文窗口（8192 tokens），并使用改进的 RoPE 位置编码。

**对齐方法**：Llama 3 Instruct 版本经过 SFT（监督微调）、RLHF（包含拒绝采样和 PPO）和 DPO 的多阶段对齐训练，注重在帮助性和安全性之间取得平衡。

## 评测结果

报告在多个标准基准上展示了 Llama 3 的评测结果：
- **MMLU**：Llama 3 70B 达到约82%，接近当时闭源模型水平
- **HumanEval**：代码生成能力显著提升（约82%）
- **MATH**：数学推理能力大幅改善
- **ARC Challenge、HellaSwag、Winogrande** 等常识推理基准均达到开源模型最优水平

## 对开源 LLM 评测生态的影响

Llama 3 的发布显著缩小了开源与闭源模型的评测差距，推动了开源 LLM 评测研究的进一步普及。其详细的评测方法论和训练细节公开，成为后续开源模型研究的重要参考基准。

## 开放许可

Llama 3 采用开放许可（Meta Llama 3 Community License），允许商业使用，进一步扩大了其作为评测基准参照系的影响力。

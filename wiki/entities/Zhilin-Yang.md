# Zhilin Yang（杨植麟）

> Moonshot AI（月之暗面 / Kimi）创始人兼 CEO。清华本科 + CMU PhD，主要工作横跨 Transformer-XL、XLNet 等基础架构和 Kimi 系列长上下文产品。截至 2026-04 Moonshot 估值已突破 200 亿美元，是中国 LLM 评测圈在长上下文 / 多模态方向最具影响力的创始人之一。

## 基本信息

- **所属机构**：Moonshot AI（2023 创立至今）
- **职位**：Founder & CEO
- **学历背景**：清华大学 BS（2015）；Carnegie Mellon University PhD（CS，导师 Ruslan Salakhutdinov、William Cohen）
- **链接**：[Wikipedia](https://en.wikipedia.org/wiki/Yang_Zhilin) · [Google Scholar](https://scholar.google.com/citations?user=DN5d-CYAAAAJ)

## 评测领域主要贡献

**Transformer-XL / XLNet（CMU 时期）**：博士期间共同发表 *Transformer-XL: Attentive Language Models Beyond a Fixed-Length Context*（ACL 2019）和 *XLNet*（NeurIPS 2019），后者在 GLUE / SQuAD / RACE 等基准上超越 BERT，是 BERT 时代评测范式的关键变革之一——也是其后续在长上下文方向深耕的学术根。

**Kimi 长上下文评测**：Moonshot 自 2023-10 推出 Kimi Chat 起即把"长上下文"作为差异化定位，先后支持 200K / 1M / 2M token，对国内 [[long-context-eval|长上下文评测]] 议题（[[RULER]] / [[NIAH]] / [[LongBench]] / [[LongBench-v2]]）形成实际推动。

**Kimi K2 / K2.5 开源**：2025-2026 期间 Moonshot 推出 Kimi K2、K2.5 等开源模型，在 [[HumanEval]] / [[MMLU]] / [[GPQA]] / [[AIME]] 等评测上取得与 GPT-5.x 同档表现（K2.6 在编码评测上 ties GPT-5.5）。Kimi 系列是中文 LLM 评测圈最有竞争力的开源选项之一。

**MoonViT 多模态视觉编码器**：2026-01 Kimi K2.5 引入 400M 参数的 MoonViT 视觉编码器，原生多模态评测能力对标 [[Qwen-VL]] / GPT-4V 系列。

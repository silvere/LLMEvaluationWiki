# Ilya Sutskever

Ilya Sutskever 是加拿大计算机科学家，Safe Superintelligence Inc.（SSI）的联合创始人，曾担任 OpenAI 联合创始人和首席科学家（2015-2024年）。他是现代深度学习和大型语言模型发展历程中的核心人物之一。

## 学术背景

Sutskever 在多伦多大学 Geoffrey Hinton 的指导下获得博士学位。他与 Hinton 及 Alex Krizhevsky 合作开发的 AlexNet（2012年）在 ImageNet 竞赛中取得突破性成绩，标志着深度学习时代的正式开启。

## 在 OpenAI 的工作

Sutskever 是 OpenAI 的七位联合创始人之一，担任首席科学家期间主导了多项核心技术方向。他是 GPT 系列语言模型（包括 GPT-2、GPT-3）的主要研究贡献者之一，并领导 OpenAI 团队开展了与 GPT-4 及 InstructGPT 相关的研究工作。其领导的 Superalignment 团队致力于探索超级智能的对齐方法论。

## 与 LLM 评测的关联

Sutskever 对 LLM 能力边界和扩展规律（Scaling Laws）有深入研究，这些工作直接影响了评测体系的设计方向：如何选择能真实反映模型能力跃迁的评测任务，以及在何种规模下特定能力会涌现（Emergent Capabilities）。他公开表达了对 LLM 真正理解能力与表面统计记忆之间差异的关注。

## SSI（Safe Superintelligence Inc.）

2024年，Sutskever 与 Daniel Gross 和 Daniel Levy 联合创立 SSI，专注于以安全为首要目标的超级智能研究。SSI 的工作方向与 AI 能力评测和安全评估密切相关，但具体研究内容尚未大规模公开发表（截至本文写作时）。

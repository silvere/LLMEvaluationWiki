# SWE-bench Multimodal: Do AI Systems Generalize to Visual Software Domains?

> [[SWE-bench]] 多模态扩展（Princeton + Stanford + Meta，2024-10）。**617 个 JavaScript / UI / visualization 任务**，每个含至少 1 张图片。揭示「SWE-bench Python 强的模型在多模态前端任务上崩塌」：SWE-agent 仅 12% 通过，对照组 6%。

## 论文核心贡献

1. **首个跨语言 + 多模态 SWE benchmark**：从 Python → JavaScript + 视觉
2. **617 任务 × 17 JS 库**：web design / diagramming / visualization 等真实 UI 库
3. **强制视觉理解**：每个 task 至少 1 张图（UI 截图 / 设计稿）
4. **揭示泛化失败**：Python SOTA 不等于 JS SOTA，视觉推理是关键短板

# TravelPlanner: A Benchmark for Real-World Planning with Language Agents

> 复旦 + OSU + PSU + Meta 2024-02 推出的 **language agent 真实世界规划评测**（ICML 2024 Spotlight）。**1225 旅行规划意图 + ~400 万数据条 sandbox**，GPT-4 成功率仅 0.6%，揭示当前 agent 在长跨度 + 多约束规划任务上的根本困境。

## 论文核心贡献

1. **1225 精心人工策划规划任务**：日程 / 预算 / 航班 / 酒店 / 餐饮多维约束
2. **~400 万数据 sandbox**：航班 / 酒店 / POI / 餐厅真实数据
3. **GPT-4 仅 0.6%** 成功率：揭示「task consistency / tool selection / constraint management」三大短板
4. **多约束求解评测体系**：硬约束（预算）+ 软约束（偏好）的可验证 evaluation

## 评测圈意义

- 是 [[GAIA]] / [[AgentBench]] 之外**面向复杂规划**的 agent 评测
- 多约束求解能力是 agent 进入实际生产的关键
- 与 [[Mind2Web]] / [[WebArena]] 形成「规划 + 操作」双轨

## 限制 / pitfall

- 数据 sandbox 是静态快照，与真实可变库存有 gap
- 0.6% 成功率太低，区分度不足以排名前沿模型
- 后续 agent + tool-use 提升后需扩规模或难度

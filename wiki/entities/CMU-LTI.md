# CMU LTI（卡内基梅隆大学语言技术研究所）

CMU LTI（Language Technologies Institute）是卡内基梅隆大学（Carnegie Mellon University）计算机科学学院下设的研究所，成立于1996年，是全球历史最悠久、影响力最大的自然语言处理专业研究机构之一。

## 研究方向

LTI 的研究覆盖自然语言处理的广泛领域，包括机器翻译、信息检索、语音识别、对话系统、文本摘要以及语言模型评测。研究所拥有数十位教授和数百名研究生，每年在顶级 NLP 会议（ACL、EMNLP、NAACL 等）发表大量论文。

## 多语言 NLP 评测贡献

CMU LTI 在多语言 NLP 评测领域有深厚积累。研究人员参与构建了多个跨语言评测基准，探讨低资源语言的迁移学习能力评测方法。Graham Neubig 等教授主导的研究关注多语言模型在零样本跨语言迁移场景下的性能评测，涵盖语言理解、问答和生成等任务。

## 对话系统评测

LTI 长期从事对话系统研究，包括任务型对话（Task-oriented Dialogue）和开放域对话（Open-domain Dialogue）的评测方法。研究人员探索了人工评测与自动指标结合的混合评测框架，针对一致性（Coherence）、信息量（Informativeness）、流畅性（Fluency）等多维度开发了评测标准。

## 代表性研究人员

CMU LTI 培养了众多 NLP 领域的杰出学者，包括多位现任于主要 AI 研究机构的研究员。实验室与工业界（Google、Microsoft、Meta 等）保持密切合作，推动学术评测成果向产业应用转化。

## 与 LLM 评测的关联

随着大型语言模型的兴起，CMU LTI 的研究人员积极参与 LLM 能力边界探索，包括 LLM 在机器翻译质量评估中的应用（使用 LLM 替代人工评测）、多语言推理能力评测以及代码生成评测等新兴方向。

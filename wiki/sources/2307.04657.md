# BeaverTails: Towards Improved Safety Alignment of LLM via a Human-Preference Dataset

## 论文概述

BeaverTails 是北京大学等机构联合提出的 LLM 安全对齐数据集与评测框架，发表于2023年7月。该工作旨在通过构建高质量的人类安全偏好数据集，推动 LLM 安全对齐研究的标准化和可比较性。

## 数据集构建

**规模**：BeaverTails 包含约330,000条问答对，每对均附有"安全/不安全"的人类标注，以及在"安全性"和"帮助性"两个维度上的相对偏好对比标注。

**危害分类体系**：定义了14种危害类别（Harm Categories），覆盖：
- 仇恨言论（Hate Speech）
- 暴力内容（Violence）
- 自我伤害（Self-harm）
- 隐私侵犯（Privacy Violation）
- 违法活动辅助（Illegal Activities）
- 误导信息（Misinformation）等

**问题来源**：基于 Alpaca、ShareGPT 等数据集的真实用户指令，确保评测的现实性。

## 评测贡献

BeaverTails 提供了：
- 用于训练安全奖励模型（Safety Reward Model）的偏好数据
- 用于评测 LLM 安全对齐效果的标准测试集
- 安全性与帮助性之间权衡的量化分析框架（Safe-Helpful Tradeoff）

## 主要发现

- 提高安全性和提高帮助性存在内在张力，但并非完全对立
- 现有 RLHF 训练的模型在某些危害类别上仍有明显漏洞
- 细粒度的危害分类有助于精准诊断安全对齐的薄弱环节

## 学术影响

BeaverTails 为 LLM 安全评测提供了细粒度的危害分类框架和大规模偏好数据，是安全对齐研究领域的重要基础资源之一。

# WebArena: A Realistic Web Environment for Building Autonomous Agents

## 论文概述

WebArena 是由卡内基梅隆大学等机构提出的网络代理评测基准，发表于2023年7月。该基准通过构建独立的真实网站环境（而非公开互联网），系统评测 LLM 代理在真实网络交互任务中的端到端执行能力。

## 评测设计

WebArena 的核心创新在于构建了一套**完全自托管、可复现的真实网站环境**，包括：

- **Reddit 克隆**（Postmill）：社区讨论、发帖、评论
- **GitLab 克隆**：代码仓库管理、Issue 追踪、Pull Request
- **电商网站**（OneStopShop）：商品搜索、购买、评价
- **地图服务**（OpenStreetMap）：路径规划、地点搜索
- **内容管理系统**（CMS/Wikipedia 克隆）

共包含约800个长程任务（Long-horizon Tasks），每个任务需要代理执行多步网页操作（点击、输入、导航）才能完成。

## 评测指标

- **任务成功率**（Task Success Rate）：基于功能验证而非文本匹配
- **路径效率**（Path Efficiency）：完成任务所需步骤数与最优路径的比较

## 主要发现

- 最佳 GPT-4 代理在 WebArena 上的任务成功率约为14%，远低于人类水平（78%）
- 真实网站环境比合成任务更好地揭示了 LLM 代理的规划和执行能力局限
- 长程任务中的错误积累（Error Compounding）是代理失败的主要原因

## 影响

WebArena 推动了网络代理评测的规范化，其自托管环境设计成为后续代理基准的参考范式。

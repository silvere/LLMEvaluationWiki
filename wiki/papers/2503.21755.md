# VBench-2.0: Advancing Video Generation Benchmark Suite for Intrinsic Faithfulness

> S-Lab NTU + 上海 AI Lab + 中大 + 港大 2025-03 推出的 [[VBench]] 第三代。**评测从表面感知质量推向「内在保真度」**：物理定律、人体解剖、常识合理性是否成立。集成 SOTA VLM + 异常检测专家 + 人工标注。

## 论文核心贡献

1. **5 维度内在保真度评测**：
   - **Human Fidelity**（人体解剖正确性）
   - **Controllability**（对 prompt 的遵从度）
   - **Creativity**（创造力 / 多样性）
   - **Physics**（物理规律合理性）
   - **Commonsense**（常识合理性）
2. **混合评测管线**：VLM generalist + anomaly detection specialist + human validation 三层
3. **面向高阶应用**：AI 辅助电影 / 世界模拟，比 v1 (16 dims) / v++ (T2V+I2V) 更注重「真不真」
4. **揭示 Sora / Veo 类模型在 Physics + Commonsense 子档上的代际差距**

## 演进路径

- **VBench v1** (2023, 16 dims)：感知质量分解
- **VBench++** (2024, arxiv:2411.13503)：扩展 T2V + I2V 公平对比
- **VBench-2.0** (2025)：**5 维内在保真度** —— 评测从「好看」转向「合理」

## 限制 / pitfall

- 「Physics 合理性」高度主观，标注一致性是关键
- VLM 评分器自身的 capability bottleneck（VLM 不懂 Physics 就评不准）
- 计算成本高（VLM + 异常检测双管线）

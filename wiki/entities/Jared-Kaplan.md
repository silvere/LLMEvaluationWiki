# Jared Kaplan

> Anthropic 联合创始人，首席科学官（CSO）兼 Responsible Scaling Officer（2024-10 起）；理论物理学家出身，*Scaling Laws for Neural Language Models* 主要作者，建立模型评测「数据×参数×算力」可预测性的奠基者。

## 基本信息

- **所属机构**：Anthropic（联合创始人）·Johns Hopkins University（物理与天文系副教授）
- **职位**：Chief Science Officer + Responsible Scaling Officer
- **学历背景**：Stanford BS（物理 + 数学）；Harvard PhD（物理）
- **链接**：[Wikipedia](https://en.wikipedia.org/wiki/Jared_Kaplan) · [Anthropic](https://www.anthropic.com/)

## 评测领域主要贡献

**Scaling Laws 奠基**：2019 加入 OpenAI 期间，与 Sam McCandlish 等共同主导 *Scaling Laws for Neural Language Models*（2020），首次系统量化「test loss 与模型规模、数据量、计算预算」的幂律关系。这一框架成为后续所有大模型训练前的「评测可行性」论证基础——决定该不该训、要训多大。

**Responsible Scaling Policy（RSP）**：在 Anthropic 主导建立的 RSP 是行业首个把安全评测门槛与模型能力等级直接挂钩的公开框架（AI Safety Levels, ASL-2/3/4），定义了在每个能力等级下需要通过哪些安全评测才能 release。是 [[safety-eval-landscape]] / [[capability-elicitation]] 等评测协议的工业级实践范例。

**Anthropic 模型评测体系**：作为 CSO 主导 Claude 系列（Claude / Claude 2 / Claude 3 / Claude 3.5 / Claude Sonnet 4.5 / Opus 4 等）的训练科学和评测协议设计，定义了 Anthropic system card 中的评测维度组合（reasoning、agentic、coding、CBRN safety 等）。

# 基准效度（Benchmark Validity）

> 评测基准是否真正测量了其声称的目标能力的方法论问题，是评测可信度的根基，区分"好成绩"与"真实能力"的关键。

## 定义

效度（Validity）是心理测量学的核心概念，在 LLM 评测语境中指：基准分数是否真的反映了目标能力，而不是其他混淆因素。

**效度的主要类型**：
- **构念效度（Construct Validity）**：基准分数与理论上的目标能力构念的对应程度
- **内容效度（Content Validity）**：基准任务是否覆盖了目标能力的全部重要方面
- **预测效度（Predictive Validity）**：基准分数能否预测模型在真实应用中的表现
- **区分效度（Discriminant Validity）**：基准是否能区分真正具有该能力和只是擅长该基准形式的模型

## 重要性（在 LLM 评测中）

效度危机是当前 LLM 评测面临的最深层方法论问题：

1. **捷径学习（Shortcut Learning）**：高分模型可能利用数据集偏差而非目标能力（如通过选项长度猜测 MMLU 答案）
2. **格式 vs 能力**：多项选择格式的高分可能反映格式熟悉度而非知识理解
3. **表面相关性**：高 BLEU 分数不一定意味着高翻译质量；高 ROUGE 不一定意味着好摘要
4. **能力迁移验证**：真正有效的能力评测，基准分数应能预测相关实际任务的性能

## 主要方法/实现

**效度评估方法**：

**相关性分析**：
```python
# 计算基准分数与"黄金标准"（真实能力代理）的相关性
from scipy.stats import spearmanr
# 如：BLEU 分数与人类翻译质量评分的 Spearman 相关
corr, p_value = spearmanr(bleu_scores, human_quality_scores)
```

**捷径检测**：
- 设计"对抗样本"，使捷径策略失效，检验高分模型是否性能大幅下降
- 分析模型错误模式，检查是否集中于需要真实理解而非模式匹配的题目

**跨基准一致性**：声称测量相同能力的不同基准，应在同一批模型上产生类似的排名（若差异大，至少一个效度有问题）

**HELM 的多维评测框架**：
- 在同一任务类型下使用多个数据集
- 报告任务级别而非基准级别的能力估计
- 通过多角度验证效度

## 局限与挑战

- **黄金标准的循环性**：验证基准效度需要更可靠的外部标准，但"真实能力"本身难以独立测量
- **任务自然性争议**：标准化测试任务（如 MMLU 的选择题）天然是人工的，现实任务很少这样呈现
- **效度随时间的退化**：模型为基准优化后，原本有效的基准可能失效（[[evaluation-overfitting]]）
- **文化和语言效度**：以英文为中心设计的基准在非英文场景中效度存疑
- **主观任务的效度标准**：对创意写作等任务，"正确"的效度标准本身就有争议

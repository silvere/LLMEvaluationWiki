# MMLU-ProX: A Multilingual Benchmark for Advanced Large Language Model Evaluation

> 多机构联合 2025-03 推出的 [[MMLU-Pro]] 多语言扩展。**29 种语言 × 11829 完全相同题目**，可做严格 cross-linguistic 对比。Lite 版每语言 658 题。36 SOTA 模型实测，**high-resource 与 low-resource 语言差距高达 24.3 pt**。

## 论文核心贡献

1. **29 语言（含 typologically diverse）覆盖**：超越 GlobalMMLU 的 13 语言
2. **11829 严格平行题目**：完全相同的题面，仅语言不同 → 真正的跨语言对比
3. **专家审校**：每条翻译经人工校验
4. **36 模型实测**：覆盖开源 + 闭源 + 中英双优势模型
5. **量化语言不平等**：24.3 pt 资源差，明确多语言 LLM 改进的方向

## 与 MMLU-Redux / Global-MMLU 的关系

- [[MMLU-Redux]]：英文 MMLU 标签纠错（5700 题）
- Global-MMLU：13 种语言扩展（Cohere for AI）
- **MMLU-ProX**（本文）：29 种语言 + 基于 MMLU-Pro 难度

# Shanghai Jiao Tong University（SJTU，上海交通大学）

> 中国顶尖研究型大学之一，AI 与 NLP 研究在国内位列前列。在 LLM 评测圈贡献的研究者含 [[Pengfei-Liu]]（评测方法论方向）等。

## 与 LLM 评测相关的主要 group

- **电子信息与电气工程学院（SEIEE）**：[[Pengfei-Liu]] 等 AI / NLP 研究者所在
- **致远学院**：跨学科创新人才培养
- **AI Institute**：跨院系 AI 研究

## 在 LLM 评测圈的贡献

**评测方法论研究**：以 Pengfei Liu 为代表，长期关注 prompt engineering / fine-tuning / LLM-as-Judge 等评测可靠性议题，多次合作发表中文 LLM 评测综述。

**中文 benchmark 实践参与**：在 [[C-Eval]] / [[CMMLU]] / [[AGIEval]] 等中文评测基准的实测、改进与方法学讨论上有持续贡献。

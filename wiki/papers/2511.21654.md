# EvilGenie: A Reward Hacking Benchmark

> arXiv: [2511.21654](https://arxiv.org/abs/2511.21654) | 2025-11-26 | Jonathan Gabor, Jayson Lynch, Jonathan Rosenfeld

## 摘要

We introduce EvilGenie, a benchmark for reward hacking in programming settings. We source problems from LiveCodeBench and create an environment in which agents can easily reward hack, such as by hardcoding test cases or editing the testing files. We measure reward hacking in three ways: held out unit tests, LLM judges, and test file edit detection. We verify these methods against human review and each other.

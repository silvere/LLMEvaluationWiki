# TheAgentCompany: Benchmarking LLM Agents on Consequential Real World Tasks

> CMU + Duke + OpenHands 2024-12 推出的**模拟软件公司 agent 评测**。**自包含环境内含 internal websites + 数据**，模拟小型软件公司运转。175 个真实任务，**最强模型 30% 完成率（v3）**，长 horizon 任务仍超出当前 agent 能力上限。

## 论文核心贡献

1. **完整公司环境模拟**：GitLab + 文档站 + 内部聊天 + 数据库 + 财务系统
2. **175 任务覆盖多角色**：工程师 / PM / 财务 / 行政 / 销售
3. **autonomous task completion 度量**：从无人监督完成度评估
4. **30% 任务完成率**（Claude-3.5-Sonnet v3 更新后；论文 v1 为 24%）
5. **长 horizon 仍是上限**：>50 步的任务普遍失败

## 评测圈意义

- 是 [[GAIA]] / [[SWE-bench-Verified]] 之外**面向「替代人类员工」**的 agent 评测
- 「sandbox 内部网站」范式被 [[WebArena]] / [[OSWorld]] 借鉴
- 与 [[SWE-Lancer]] 形成「软工 agent 经济价值评测」双轨

## 限制 / pitfall

- 模拟环境，与真实企业仍有 gap
- 175 任务，更新跟进 frontier 模型需要持续投入
- 长 horizon 任务的 step-level 评分尚无标准

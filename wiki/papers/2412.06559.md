# ProcessBench: Identifying Process Errors in Mathematical Reasoning

> 阿里 Qwen 团队 2024-12 推出的**数学推理过程级错误定位**评测（ACL 2025）。**3400 测试用例**覆盖竞赛 / 奥赛级题目，每条都有人工标注的 step-level 错误位置，揭示「现有 PRM 普遍无法泛化到 GSM8K/MATH 以外的难题」。

## 论文核心贡献

1. **首个大规模 step-level 错误检测 benchmark**：3400 题，竞赛 / Olympiad 级
2. **人工标注 step-level 错误位置**：可量化 PRM 的 fine-grained 定位精度
3. **揭示 PRM 严重 OOD 泛化困境**：现有 PRM 训练时只见 GSM8K/MATH，过难题就崩
4. **QwQ-32B-Preview 与 GPT-4o 持平**，但都不及 o1-mini → o1 类推理模型在 PRM 任务上代际碾压

## 方法 / 数据集

- 数据：竞赛 / 奥赛级数学题 × 每题多个 LLM 解题轨迹
- 标注：人工标注每一步是否正确 + 首个错误 step 位置
- 指标：error step localization F1 / accuracy

## 限制 / pitfall

- 标注主观性：什么算「错误 step」存在边界争议
- **不评测 final answer 正确性**，只评 process
- 与 [[PRMBench]] 互补：ProcessBench 偏 math，PRMBench 偏多维度

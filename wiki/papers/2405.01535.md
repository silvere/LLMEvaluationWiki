# Prometheus 2: An Open Source Language Model Specialized in Evaluating Other Language Models

> [[Prometheus]] 续作（KAIST + CMU + MIT + AI2 + LG AI Research，EMNLP 2024）。通过 **weight merging** 把「直接打分」和「pairwise 偏好」两类判别能力融到一个模型里，是当前**与人类 / GPT-4 judge 相关性最高的开源 evaluator LM**。

## 论文核心贡献

1. **统一两种判别格式**：direct assessment + pairwise ranking 在一个模型内（先分头训，再 weight-merge）
2. **支持 user-defined 评测 rubric**：不仅是 helpfulness 之类通用维度
3. **8 个 benchmark 全部刷 SOTA**（开源 evaluator LM 内）
4. **Pearson r 与 GPT-4 judge / human ≥ 0.6**（v1 仅 0.3-0.5）

## 方法 / 数据集

- 起点：Mistral-7B + Mixtral-8x7B
- 训练数据：[[Feedback-Collection]]（direct）+ Preference-Collection（pairwise）
- 关键 trick：分别微调两个变种 → linear weight merging（无需联合训练）

## 限制 / pitfall

- judge 能力受限于 Mixtral-8x7B 的基础能力（数学/code 较弱）
- 在「judge 自身偏好」存在场景仍有偏差
- 与 [[RewardBench]] / [[JudgeBench]] 评估指标各异，结果不可直接对齐

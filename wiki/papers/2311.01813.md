# FETV: A Benchmark for Fine-Grained Evaluation of Open-Domain Text-to-Video Generation

> 北大 + MSR 2023-11 推出的 **T2V 细粒度评测**（NeurIPS 2023 D&B Track）。沿 3 个独立维度组织 prompt：content type / controllable attributes / complexity level，**实证 CLIPScore + FVD 与人类判断相关性较低**，提出改进指标。

## 论文核心贡献

1. **3 维 prompt 分类法**：content type × controllable attributes × complexity level
2. **手工评估 4 个代表模型**：完整人类标注 baseline
3. **打破「CLIPScore + FVD 是金标准」迷思**：与人类相关性差
4. **提出改进自动指标**：与人类判断对齐显著提升

## 限制 / pitfall

- 评测 4 模型规模较小（2023 年代际限制）
- 后续被 [[VBench]] / [[EvalCrafter]] 在维度数和模型覆盖上超越
- 仍是「T2V 自动指标不可靠」的重要参考论文

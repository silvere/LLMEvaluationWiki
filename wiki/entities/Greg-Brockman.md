# Greg Brockman

> OpenAI 联合创始人、总裁兼董事长；2026 年 5 月正式接管 OpenAI 全部产品战略，将 ChatGPT、Codex、Developer API 合并为统一组织，主导「agentic future」消费与企业级产品方向。

## 基本信息

- **所属机构**：OpenAI（2015 联合创始）
- **职位**：President, Chairman & Co-Founder。2026-05-16 起正式接管所有产品战略（此前以代理身份覆盖 Codex/ChatGPT 期间 CEO of AGI Deployment Fidji Simo 医疗假，5 月后转为永久任命）
- **学历背景**：Harvard、MIT 就读期间辍学创业；早年 CTO of Stripe
- **链接**：[个人主页](https://gregbrockman.com/) · [LinkedIn](https://www.linkedin.com/in/thegdb) · [Wikipedia](https://en.wikipedia.org/wiki/Greg_Brockman)

## 评测领域主要贡献

**GPT 系列训练工程化与评测发布**：作为 OpenAI 联合创始人和首任 CTO，主导 GPT-3 / GPT-4 训练 infra 与评测协议制定。GPT-4 Technical Report 中关于 MMLU / HumanEval / 多语言评测的报告结构成为后续行业事实模板。

**评测协议公共化**：长期推动 OpenAI 评测结果以 system card / model spec 形式公开发布；OpenAI eval framework（早期 [evals](https://github.com/openai/evals) 开源仓库）由其牵头建立，是 LLM 公开评测可复现实践的早期标杆。

**Agent / 推理产品评测整合**（2026）：当前在 OpenAI 内部主导将 ChatGPT、Codex、Developer API 合并为统一产品线，意味着 OpenAI 评测体系也将向「agent-first」整合——这会显著影响 [[tau-bench]] / [[OSWorld]] / [[SWE-bench-Verified]] 等 agent 评测在 product context 中的优先级。

## 代表性工作

- OpenAI Gym（2016，开源 RL 评测平台）
- OpenAI Evals 框架（2023 开源）
- GPT-3 / GPT-4 / o1 / o3 等模型 launch
- ChatGPT 早期发布（2022-11）

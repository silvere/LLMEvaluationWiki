# XLNet（1906.08237）

> [[Zhilin-Yang]] / [[Quoc-Le]] / Salakhutdinov 等 2019-06 发表的 generalized autoregressive 预训练模型。结合 BERT 的双向上下文与 GPT 的自回归生成，通过 permutation language modeling 学习。在 SQuAD / GLUE / RACE 等 20+ 任务上一度全面超越 BERT。是 2019 年 NLP 评测的里程碑论文。

## 核心贡献

- **Permutation Language Modeling**：随机排列序列的 factorization order，让自回归模型也能学到双向依赖
- **Two-Stream Attention**：解耦 content 和 query 表示，解决 permutation 训练的 target 泄漏
- **Transformer-XL backbone**：长上下文 attention，segment recurrence + relative position encoding

## 评测影响

- 一度在 GLUE / SuperGLUE / SQuAD 全面超越 BERT，开启 2019-2020 NLP benchmark 竞赛
- 推动 RoBERTa / T5 / ELECTRA 等后续预训练模型的方法学反思（数据规模 vs 训练目标的相对重要性）
- 「permutation LM」思想后续在多种小众模型继续出现

## 历史定位

- 被 RoBERTa（同年 2019-07）超越，揭示「训练数据 + 训练时长」比训练目标本身更重要
- 是 transformer 预训练「百花齐放」时期的代表作
- Quoc Le / Zhilin Yang 后续 work 多次延续此思路

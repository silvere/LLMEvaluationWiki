# Evaluation Agent: Efficient and Promptable Evaluation Framework for Visual Generative Models

> S-Lab NTU + 上海 AI Lab 2024-12 推出的 **agent 驱动视觉生成自动评测框架**。用 human-like 多轮策略「按需采样」评测，**计算成本降到传统方法 10%**，支持用户 prompt 化定制评测目标。

## 论文核心贡献

1. **Agent 驱动多轮评测**：agent 根据已有样本决定下一步采哪个 prompt，类似 active learning
2. **10x 算力效率**：少量样本即可逼近 traditional benchmark 的结果
3. **可 prompt 化的评测目标**：用户用自然语言描述「我想评什么」，agent 设计 evaluation pipeline
4. **可解释**：输出分数 + 推理过程，不只是数字

## 评测圈意义

- 是 [[VBench]] 团队的「下一代评测理念」：从 static benchmark → adaptive evaluation agent
- 适合**新模型快速 screening**：不用跑完整 VBench，10 分钟内出初步评估
- 与 [[Petri]] 等 safety auditing agent 框架共享「评测 = 自主 agent」范式

## 限制 / pitfall

- agent 自身的判断力上限决定评测可靠性
- 跨模型对比时如果 agent 策略不同则不可比
- 重现性依赖 agent 随机种子稳定

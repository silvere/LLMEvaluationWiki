# Pengfei Liu（刘鹏飞）

> 上海交通大学副教授，致远学院 / SEIEE。中国 LLM 评测方法论方向的代表青年研究者，长期关注「评测可靠性 / 生成评测 / 预训练对评测的影响」。

## 基本信息

- **所属机构**：Shanghai Jiao Tong University（致远学院 / SEIEE）
- **职位**：副教授
- **学历背景**：上海交通大学 PhD
- **链接**：[主页](http://pfliu.com/) · [Google Scholar](https://scholar.google.com/citations?user=oIz_CYEAAAAJ)

## 评测领域主要贡献

**Prompt-based Learning Survey**：合著 *Pre-train, Prompt, and Predict: A Systematic Survey of Prompting Methods in Natural Language Processing*（2021）—— 早期对 prompting 评测方法论的系统综述，引用过万，影响 prompt engineering 与 in-context learning 评测的方法学。

**生成评测方法论**：长期关注开放式生成的评测可靠性问题，包括 LLM-as-Judge 的偏差、reference-free metric、human-AI evaluation 一致性等议题。

**LLM 评测框架与基准**：参与多个 LLM 评测基础设施与基准数据集设计，对中文 LLM 评测开源生态贡献显著。

**Generative Judge / Self-Evolving Eval**：探索"模型自身演化评测协议"——评测随模型能力提升而动态调整，是 dynamic benchmark 议题的方法学线索之一。

## 代表性工作

- 多个 LLM evaluation 方法论论文
- SJTU 开源评测基础设施

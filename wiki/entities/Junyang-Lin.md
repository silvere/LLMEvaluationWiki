# Junyang Lin（林俊旸）

> 阿里 Tongyi Qwen 大模型团队前技术负责人；2026 年 3 月离开阿里创业，新公司估值约 $2B，聚焦 world models 与 embodied AI。是中国大模型评测圈在国际舞台最具影响力的工程负责人之一。

## 基本信息

- **当前**：自创公司创始人（2026-03 起）。新公司方向为 world models 和 embodied brains，已招募来自 ByteDance、Tencent 等团队成员，启动融资估值约 20 亿美元
- **前职位**：Alibaba DAMO Academy P10（阿里最年轻的 P10 之一）；Tongyi Qwen 技术负责人。2019 起入职阿里，主导 Qwen 系列从 0 到 1 的开源发布，全球下载量超 6 亿次
- **学历背景**：北京大学 语言学与应用语言学 硕士
- **离任背景**：2026-03-04 凌晨在社交媒体宣布离职。媒体报道指阿里重组 Qwen 团队，将预训练、后训练、文本、多模态拆分为不同小组，管理范围与其团队整合理念冲突
- **链接**：[Google Scholar](https://scholar.google.com/citations?user=cnyDC2EAAAAJ) · [Bloomberg 报道](https://www.bloomberg.com/news/articles/2026-03-04/alibaba-qwen-head-who-warned-of-openai-gap-steps-down)

## 评测领域主要贡献

**Qwen 评测体系建设**：作为 Qwen / Qwen-1.5 / Qwen2 / Qwen2.5 / Qwen3 系列负责人，主导这些模型在 [[MMLU]] / [[GSM8K]] / [[HumanEval]] / [[C-Eval]] / [[CMMLU]] 等基准上的评测组织与结果发布。Qwen 系列是中文 LLM 在国际评测榜单上最有竞争力的开源选项。

**多语言 / 中文评测推动**：在 Qwen 团队推动建立面向中文及亚洲语言的评测标准，包括对 [[C-Eval]] / [[CMMLU]] / [[CMMMU]] 等中文基准的 baseline 实测与方法论贡献。

**开源大模型评测可复现性**：Qwen 系列在 Hugging Face 上的 model card + 评测代码 + 数据混合配比 的开源程度高于多数封闭厂商，对开源 LLM 评测的可复现实践形成重要参考。

**「OpenAI Gap」公开警示**：2026 年初公开发言指出中国大模型与 OpenAI 仍有显著差距，是少数从内部一线主动揭示评测真实状态的中国研究 lead。

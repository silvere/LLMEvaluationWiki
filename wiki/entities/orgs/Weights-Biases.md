# Weights & Biases (W&B)

> MLOps 与 AI 平台公司，旗舰产品为 **wandb**（ML 实验追踪）和 **[[Weave]]**（LLM 应用 observability）。成立于 2018 年，服务 OpenAI / DeepMind / NVIDIA 等顶级 AI 机构，2023 年估值超 $10 亿。

## 主要产品

- **W&B（wandb）**：训练实验追踪 / 超参数优化 / 模型 registry
- **[[Weave]]**：LLM 应用 tracing + evaluation + playground

## 评测圈意义

- 在 AI 研究训练侧占据统治地位，Weave 是其向 LLM 评测延伸的战略产品
- 与 [[Langfuse]] / [[LangSmith-Eval]] / [[Arize-Phoenix]] / [[Braintrust]] 并列 LLM observability 主流

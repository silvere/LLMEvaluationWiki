# Cohere

> 加拿大企业级 AI 公司，Command 系列模型的开发者，专注于企业 NLP 应用，由前 Google Brain 研究员创立。

## 基本信息

- **性质**：营利性 AI 公司
- **成立时间**：2019 年
- **总部**：加拿大多伦多
- **联合创始人**：Aidan Gomez、Ivan Zhang、Nick Frosst（前 Google Brain）
- **规模**：数百人

## 主要贡献（评测相关）

**MTEB 基准贡献**：Cohere 研究团队是 MTEB（Massive Text Embedding Benchmark，2022）的主要作者之一。MTEB 涵盖 56 个数据集和 8 类评测任务（分类、聚类、重排序、检索等），成为文本嵌入模型评测的行业标准。

**Command 系列评测**：Cohere 持续发布 Command 系列模型在企业相关评测集（长文档理解、RAG 质量、工具调用准确性等）上的性能报告，为企业级 LLM 评测提供了商用参照。

**Embed 模型评测**：Cohere Embed 系列在 BEIR（信息检索评测基准）和 MTEB 上持续发布对比评测结果，是嵌入模型排行榜的重要参与者。

**工具调用评测**：随着 RAG 和 Agent 评测需求增长，Cohere 在工具调用准确性和 Grounded Generation（基于文档的生成）评测上发表了相关研究。

**Transformer 架构贡献**：Aidan Gomez 是 Transformer 架构原论文（"Attention Is All You Need"，2017）的共同作者之一，对后续所有 Transformer 基础评测范式均有间接奠基作用。

## 代表性模型/产品

- **Command / Command R / Command R+**：企业级指令跟随模型
- **Embed**：文本嵌入模型
- **Rerank**：重排序模型
- **Aya**：多语言开源模型

## 对评测生态的影响

Cohere 在企业级评测维度（延迟、成本效率、企业文档理解）上的关注，补充了主流学术基准对商业应用场景评测的不足。MTEB 已成为文本表示模型评测的事实标准，被 Hugging Face 等平台采用。

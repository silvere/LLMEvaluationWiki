# FLUX

> **Black Forest Labs**（Stable Diffusion 原班人马创立）2024 年发布的开源图像生成模型系列。FLUX.1-dev 和 FLUX.1-schnell 在文图对齐、细节质量上超越 Stable Diffusion 3，成为 2024-2025 年 [[DPG-Bench]] / [[PickScore]] 等评测的强基线。

## 版本

| 版本 | 特点 | 许可 |
|---|---|---|
| FLUX.1-schnell | 极速（4 步蒸馏）| Apache-2.0 |
| FLUX.1-dev | 高质量开发版 | FLUX-1-dev non-commercial |
| FLUX.1-pro | 商业 API 版 | proprietary |

# 自我增强偏差（Self-Enhancement Bias）

> LLM 在担任裁判时，倾向于给与自身生成风格相近或由自身生成的内容打出更高分数的系统性偏差。

## 定义

自我增强偏差（Self-Enhancement Bias，也称 Self-Preference Bias）是指当 LLM 用于评测文本质量时，对自己生成的文本或具有相似生成特征（风格、措辞习惯）的文本给出更高评分，即使这些文本的客观质量并不更优。

实验证据来自 Panickssery et al. (2024) 等研究：让同一 LLM 既生成回答又担任裁判时，其自生成的回答胜率系统性高于其他模型，即使人类评测认为两者质量相当。

机制推测：
- 相似的风格/词汇选择在语义空间中更接近，裁判模型对其"感觉更好"
- 模型的内容偏好（倾向某类论点结构）影响质量判断
- 训练数据分布导致的隐性偏好

## 重要性（在 LLM 评测中）

自我增强偏差对 LLM 评测生态影响深远：

1. **跨模型公平性**：若使用 GPT-4 作为裁判评测所有模型，GPT-4 系列模型可能系统性获益，排行榜产生结构性偏差
2. **自评不可靠**：让模型评估自己的输出质量（如 self-critique 场景）会虚高估计
3. **裁判选择的政治性**：研究者在选择哪个模型做裁判时，结果可能偏向该模型的"亲属"模型
4. **复合偏差效应**：与 verbosity bias、position bias 叠加，可能显著放大某些模型的排行榜分数

## 主要方法/实现

**检测方法**：
- 使用多个不同系列的 LLM 作为裁判，比较同一对比较的裁判结论是否系统一致
- 分析裁判给自家模型的胜率是否显著高于人类裁判的结果
- 移除可识别的生成风格标记后重新评测（去身份化实验）

**缓解策略**：
- **多裁判共识**：使用多个不同系列 LLM 裁判，只保留共识结果
- **交叉评测**：避免用同一模型系列评测相关模型
- **盲评设计**：对回答来源匿名化，但 LLM 可能仍能识别风格
- **人类评测校准**：定期用人类评测验证 LLM 裁判的偏差程度

## 局限与挑战

- **难以完全去除**：风格相似性是客观存在的，要完全分离"风格偏好"与"质量判断"本身就困难
- **定义边界模糊**：部分研究认为相同风格的文本确实在某些维度上更适合该裁判的任务需求，偏差与真实偏好难以区分
- **不同领域表现不同**：事实性问题中自我增强偏差较小，主观性回答中更显著
- **缺乏统一检测标准**：不同研究的检测方法不一致，偏差的量化结果差异大

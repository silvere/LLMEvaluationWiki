# MLE-bench: Evaluating Machine Learning Agents on Machine Learning Engineering

> OpenAI 2024-10 推出的 **ML Engineering agent 评测**。**75 个 Kaggle 比赛任务**，用公开 leaderboard 建立人类基线。**o1-preview + AIDE scaffolding 在约 17% 比赛中达到铜牌水平**，验证 frontier 模型 + scaffolding 能在 ML 工程任务上 outperform 一部分人类竞赛者。

## 论文核心贡献

1. **75 Kaggle 比赛任务**：覆盖多种 ML 任务类型
2. **人类 leaderboard 基线**：用真实比赛排名作铜银金奖阈值
3. **资源 scaling 实验**：研究计算预算对 agent 表现的影响
4. **数据污染评估**：检查训练数据中是否泄露竞赛信息
5. **17% 铜牌率 (o1-preview + AIDE)**：超越多数 Kaggle 入门玩家

## 评测圈意义

- 是 [[SWE-bench-Verified]] / [[TheAgentCompany]] 之外**面向 ML 工程**的 agent 评测
- AIDE scaffolding 成为 ML agent 的标准基线
- 数据污染评估方法学借鉴自 [[benchmark-contamination]]

## 限制 / pitfall

- Kaggle 比赛数据公开 → **训练数据污染风险高**（论文已分析）
- 仅 75 任务，方差较大
- 与真实工业 ML pipeline 仍有 gap

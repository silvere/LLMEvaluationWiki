# Xipeng Qiu（邱锡鹏）

> 复旦大学计算机科学教授；OpenMOSS 团队 lead（与上海创智学院 SII + 复旦 NLP 实验室 + MOSI Intelligence 合作）。MOSS 开源大模型主创，是中国大模型评测圈的代表 PI。

## 基本信息

- **所属机构**：Fudan University 计算机科学学院 · Shanghai Innovation Institution（SII，OpenMOSS lead）
- **职位**：Professor · OpenMOSS Team Lead
- **学历背景**：复旦大学 PhD
- **链接**：[主页](https://xpqiu.github.io/)

## 评测领域主要贡献

**MOSS 系列开源模型**：2023-02 发布首版 MOSS（16B 参数对话模型），是中国早期开源大模型代表之一。后续 MOSS / MOSS-RLHF 等版本在中文评测（[[C-Eval]] / [[CMMLU]] / [[AGIEval]]）上的实测奠定了国内开源 LLM 评测协议的早期实践。

**MOSS 工具增强评测**：MOSS 早期就支持外部工具调用（搜索、计算器等），是中文 LLM 在 tool-use 评测维度的早期实践，对后续 [[ToolBench]] / [[BFCL]] 等评测在中文场景的应用有方法学影响。

**LLM 综合评测方法学**：长期关注 prompt engineering / fine-tuning evaluation 方法学，并培养出 [[Tianxiang-Sun|孙天祥]] 等专注 post-training RL 评测的学生，是复旦 NLP 评测人才的核心导师之一。

**OpenMOSS 开源生态**：作为 OpenMOSS lead，推动模型训练 / 评测代码的开源化，对中文 LLM 评测可复现性有显著贡献。

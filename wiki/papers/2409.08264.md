# Windows Agent Arena: Evaluating Multi-Modal OS Agents at Scale

> Microsoft Research 2024-09 推出的 **Windows OS agent 评测**。**150+ 多样化 Windows 任务**，自研 Navi agent 成功率 19.5% vs 人类 74.5%。支持并行评测，**20 分钟跑完整 benchmark**，是 [[OSWorld]] / [[AndroidWorld]] 之外的 Windows 平台代表。

## 论文核心贡献

1. **150+ Windows 桌面任务**：跨规划 / 屏幕理解 / 工具使用
2. **Navi agent baseline**：自研多模态 agent，19.5% 成功率
3. **大规模并行评测**：20min 跑完 benchmark
4. **74.5% 人类基线**：明确「人类 vs agent」差距

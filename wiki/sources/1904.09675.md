# BERTScore: Evaluating Text Generation with BERT

> 提出 BERTScore，利用 BERT 上下文嵌入计算候选文本与参考文本的 token 级语义相似度，比 BLEU/ROUGE 等基于 n-gram 的指标与人类判断更相关。

## 核心主张
- [CLAIM]: BERTScore 使用预训练 BERT 模型提取上下文化 token 嵌入，通过贪心匹配计算候选文本与参考文本之间的精确率（Precision）、召回率（Recall）和 F1，克服了 BLEU 等指标无法捕捉语义等价的局限。
- [CLAIM]: 在机器翻译（WMT）和图像描述生成（COCO）任务上，BERTScore 与人类判断的 Pearson 相关性约为 0.67-0.74，显著优于 BLEU（约 0.33）和 ROUGE（约 0.40-0.55）。
- [CLAIM]: BERTScore 可使用不同预训练语言模型作为骨干（BERT、RoBERTa、XLNET 等），更强的骨干模型通常带来更高的相关性。
- [CLAIM]: 通过 IDF 权重对稀少但语义重要的 token 加权，可进一步提升评测质量。

## 方法论摘要

计算流程：(1) 用预训练模型编码候选文本和参考文本，得到 token 级上下文嵌入；(2) 对每个候选 token，找到相似度最高的参考 token（贪心匹配），反之亦然；(3) 计算精确率、召回率、F1。评测在 WMT16/17/18（机器翻译人类评测数据）和 COCO（图像描述）上进行。

## 数据与结果

- 评测数据集：WMT 多年人类评测数据、COCO 描述生成
- BERTScore F1（RoBERTa-large）vs. 人类 Pearson 相关：约 0.68-0.74（WMT 依年份和语言对）
- BLEU 相关性：约 0.33-0.45
- ROUGE-L 相关性：约 0.40-0.55
- IDF 加权版本比无加权版本高约 0.01-0.03 相关性

## 局限与争议

- BERTScore 依赖基础模型的语义表示，对基础模型偏差同样敏感。
- 贪心匹配不是最优全局匹配，可能低估相似度。
- 与人类判断的相关性在开放生成任务（如摘要、对话）上不如在翻译任务上高。
- 随着 LLM 能力提升，参考文本本身的质量成为瓶颈。
